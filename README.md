# 🎮 Summer Adventure — 暑假闯关学习

沪教版 **5A/5B 英语** + 北师大版 **五年级数学** · 40 天暑假闯关 SPA

纯前端，零构建，localStorage 存数据，腾讯云智聆口语评测。

---

## 🚀 快速开始

```bash
cd summer-adventure
python3 -m http.server 8123
# 浏览器打开 http://127.0.0.1:8123
```

口语评测需要启动后端：

```bash
node server.js
# 监听 :8126，调腾讯云智聆 SOE API
```

## 🎮 游戏流程

```

```

## ⚖️ 英语权重调节

饼图四份（词汇/语法/听力/跟读），每份 ≥ 10%，总和 100%。用户拖拽饼图或点 +/- 调节，控制各关出题比例。

| 权重 | 词汇/语法 | 听力/跟读 |
|:----:|:---------:|:---------:|
| 10% | 5 题 × 2.0 分 | 4 题 × 2.5 分 |
| 20% | 8 题 × 2.5 分 | 4 题 × 5.0 分 |
| 30% | 12 题 × 2.5 分 | 4 题 × 7.5 分 |
| 40% | 12 题 × 2.5 分 | 10 题 × 4.0 分 |
| 50% | 20 题 × 2.5 分 | 5 题 × 10 分 |
| 60% | 20 题 × 3.0 分 | 12 题 × 5.0 分 |

---

## 🧮 计分规则

- **每道题固定分**（无难度/速度/连击系数），由权重计算决定
- **跟读评分：** 腾讯 SOE ≥ 50 → 该题满分，< 50 → 0 分
- **每词加分：** 每个词识别分 × 1.1 → 取整，最高 100
- **错题重做：** 做对的题分数加回总分，允许循环重做直至全对

---

## 🔄 错题重做

1. 完成后显示「重做错题，抢回积分」按钮
2. 逐题重做：
   - **选择题：** 4 选项按钮 → 反馈（正确/错误）
   - **应用题：** 文本输入框 / 分数输入框（?/? 双框）
   - **跟读题：** 「✅ 已掌握，继续」按钮
3. 正确 4 秒自动跳，错误 7 秒自动跳（带音效）
4. 重做后回到成绩页，显示最新分数
5. 支持循环重做直至全部做对

---

## 🎤 跟读/录音机制

1. 进入题目 → 1秒后自动播放原声
2. 点击「开始跟读」→ 请求麦克风 → 实时音量可视化（🎤 + Wi-Fi 三格信号）
3. 安静超过 3 秒 → "🔊 请大声朗读！"
4. 安静超过 6 秒（从未出声）→ "⏭️ 未作答"（0 分）
5. 有声音后安静 3 秒 → 朗读结束 → 提交腾讯 SOE 评测
6. 发音评分 ≥ 50 → 该题满分；< 50 → 0 分

## 🏆 主页排名

```
综合分 = 正确率 × 70% + 出勤率 × 30%
```

- 出勤按唯一日历日计算
- 出勤率 = 出勤天数 / 40，上限 100%
- 学习领奖台：前 10% → 🥇 / 再 20% → 🥈 / 再 30% → 🥉
