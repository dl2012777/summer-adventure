# 学习大冒险 Summer Adventure 🎮

米兜（5年级）的暑假40天闯关学习App · 纯前端 SPA

英语（沪教版5A/5B）+ 数学（北师大版五年级）

---

## 快速开始

用任何 HTTP 服务器打开 index.html 即可：

```bash
cd summer-adventure && python3 -m http.server 8123
```

浏览器打开 http://127.0.0.1:8123

---

## 技术栈

| 层级 | 方案 |
|------|------|
| 前端 | 纯 HTML + CSS + JavaScript（零构建，零依赖） |
| 数据 | localStorage（无需后端） |
| 音频 | Web Speech API（TTS + 语音识别） |
| 音效 | Web Audio API（程序化合成） |

## 页面结构

| 页面 | 路由 | 功能 |
|------|------|------|
| 登录页 | `#login` | 首次选头像+输入名字，之后自动登录 |
| 首页 | `#home` | 打卡信息 + 排名领奖台 + 课程入口 + 学习进度 |
| 天选择页 | `#days/en` `#days/math` | 40天网格，绿=完成/灰=未完成 |
| 游戏页 | `#game/en/1` | 5阶段闯关（31题/天） |
| 学习进度 | `#dashboard` | 英语+数学双日历 + 趋势图 + 统计卡片 |

---

## 游戏流程

每天 **31 道题**，分 5 个阶段：

| 阶段 | 题数 | 说明 |
|------|------|------|
| 词汇闯关 | 12 | 英译中/中译英/选词填空，限时15-25秒 |
| 语法迷宫 | 10 | 语法选择/改错/完成句子 |
| 听力挑战 | 3 | TTS播放→选正确答案，自动播放，可重听 |
| 跟读挑战 | 3 | TTS播放→跟读→Web Speech评分 |
| Boss关 | 3 | 综合题，分数翻倍，限时30秒 |

## 核心功能

### 积分系统
```
每题得分 = 100 × 难度系数(简单1.0/中等1.5/困难2.0)
              × 速度系数(1.0~1.3)
              × 连击系数(1.0~1.5)
Boss关: 再 × 2
打卡: 基础100分 × (1.0 + 连续天数×0.1, 上限2.0)
```

### 激励体系
- **星星 → 装备**：5⭐🎩 10⭐👓 15⭐🦸 25⭐👑 40⭐🌟
- **领奖台**：前三梯队（10%/20%/30%）

### 打卡奖励
- 每天打卡额外积分
- 连续天数越多加成越高
- 漏一天重置

### 学习进度
- 英语+数学双日历上下排列
- 绿色≥80%、橙色<80%、灰色未完成
- 点击已完成天→弹窗详情
- 点击未完成天→跳转补做，可获积分

### 错题重做
- 游戏结束后可重做错题抢回积分
- 做对的题自动移出列表

### 题库
- 英语：Day 1-5 有题（Day 1 完整31题），Day 6-40 待填充
- 数学：Day 1-3 少量题，Day 4-40 待填充
- 有 `AI出题_Prompt模板.md` 可用 AI 批量生成

```

├── styles.css              ← 深色主题（英语紫/数学橙）

```
